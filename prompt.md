너는 시니어 프론트엔드 개발자이자 UI 구현 전문가야.

React + TypeScript + Vite 기반으로
"VS Code 스타일 코드 스니펫 생성 웹앱"을 만들어줘.

## 프로젝트 목표

사용자가 VS Code에서 복사한 코드를 웹사이트에 붙여넣으면,
VS Code처럼 보이는 코드 미리보기 영역에 즉시 렌더링되는 웹앱을 만들고 싶다.

단순한 코드뷰어가 아니라,
개발자들이 SNS 공유, 문서 정리, 발표자료, 블로그 첨부용으로 사용할 수 있는
"예쁜 코드 스니펫 생성기" 수준으로 만들어줘.

---

## 핵심 기능

1. 코드 입력 textarea
2. 실시간 미리보기
3. 언어 선택 드롭다운
   - javascript
   - typescript
   - jsx
   - tsx
   - styled-components
   - mixins
   - 공통 color 변수지정
   - theme
   - json
   - bash
4. 파일명 입력 필드
5. 줄번호 표시
6. VS Code 느낌의 다크 테마 기본 적용
7. 복사 버튼
8. PNG 다운로드 버튼
9. 모바일 반응형 지원

---

## 추가 기능

1. 다크 / 라이트 테마 전환
2. 폰트 크기 조절
3. 코드 영역 padding 조절
4. 배경 스타일 선택
   - solid
   - gradient
   - subtle pattern
5. 코드 카드 width 조절
6. 긴 코드일 때 overflow 처리 자연스럽게
7. 빈 코드일 때 placeholder UI 표시

---

## UI/디자인 요구사항

1. 전체 UI는 modern 하고 실무 포트폴리오 수준으로 구현
2. 코드 미리보기 카드 상단은 VS Code 창처럼 보이게 구성
3. 좌측 상단에 mac 스타일 동그라미 버튼 3개 표시
4. 파일명 표시
5. 툴바 영역은 깔끔하게 정렬
6. 전체 배경과 카드 간 대비를 자연스럽게 구성
7. 개발자 감성이 느껴지는 UI로 만들어줘
8. 너무 장난감 같은 디자인 말고 실제 서비스 MVP 느낌으로 만들어줘

---

## 기술 요구사항

1. React + TypeScript + Vite
2. 코드 하이라이팅 라이브러리 사용
   - 우선 react-syntax-highlighter 또는 Prism 계열 중 적합한 방식 선택
3. PNG 다운로드는 html-to-image 또는 html2canvas 사용
4. 컴포넌트 분리해서 재사용 가능하게 작성
5. 상태 관리 로직은 이해하기 쉽게 구성
6. 타입스크립트 타입 명확하게 작성
7. 에러 없이 실행 가능해야 함

## 구현 세부 요구사항

1. 왼쪽에는 코드 입력 영역, 오른쪽에는 미리보기 영역
2. 모바일에서는 세로 레이아웃으로 변경
3. 입력 즉시 미리보기에 반영
4. 코드 줄 수가 많아져도 레이아웃이 깨지지 않도록 처리
5. 다운로드 시 미리보기 카드만 이미지로 저장되도록 처리
6. 복사 버튼 클릭 시 사용자 피드백 표시
7. 기본 예제 코드도 하나 넣어줘서 처음 실행했을 때 화면이 비어 보이지 않게 해줘
